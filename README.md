# Onyx

**Automated, Privacy-Centric Expense Tracking**

Onyx provides real-time expense tracking by seamlessly capturing transaction data directly from supported payment applications, eliminating the need for manual data entry.

---

### Key Features

- **Automated Transaction Capture:** Leverages Android Accessibility Services to reliably log transactions from supported payment platforms in real time.
- **Privacy-First Architecture:** Ensures complete data sovereignty. All financial data is securely stored locally on your device, with zero cloud synchronization or third-party analytics.
- **Offline Functionality:** Operates entirely offline, guaranteeing maximum performance, reliability, and security without depending on network availability.

### Technical Stack

- **Kotlin & Jetpack Compose:** Built with modern Android development standards for robust logic and a responsive, declarative UI.
- **Room:** Reliable local database persistence.
- **DataStore:** Efficient user preference management.

---

*Onyx is designed for privacy-conscious users seeking automated, seamless insights into their financial habits.*
